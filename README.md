# Legal Profession in India - Study Portal

A comprehensive, interactive study guide website for learning about the legal profession in India, with detailed notes on the Advocates Act 1961, Bar Council of India regulations, and professional ethics.

## 📚 Features

- **Comprehensive Study Notes**: Well-organized content covering:
  - Medieval & Mughal Period Judicial System
  - British Era Legal Development
  - Modern Indian Legal System
  - Advocates Act 1961 - All Key Sections
  - Bar Council of India Structure and Functions
  - Professional Ethics and Conduct
  - Disciplinary Procedures

- **30+ Multiple Choice Questions**: Practice MCQs with:
  - Instant feedback on answers
  - Detailed explanations for each question
  - Category filtering (Medieval, British Era, Advocates Act, BCI, Professional Ethics)
  - Search functionality to find specific questions

- **Interactive Learning**: 
  - Click to reveal answers
  - Visual feedback for correct/incorrect responses
  - Organized navigation between sections

## 🚀 How to Host on GitHub Pages

### Prerequisites
- GitHub account
- Git installed on your computer (or use GitHub Desktop)

### Steps to Deploy

1. **Create a GitHub Repository**
   - Go to [github.com](https://github.com) and create a new repository
   - Name it: `your-username.github.io` (for personal pages) or any other name
   - Make it public
   - Don't initialize with README, .gitignore, or license

2. **Clone the Repository**
   ```bash
   git clone https://github.com/your-username/repository-name.git
   cd repository-name
   ```

3. **Copy Website Files**
   - Copy all files from the `website` folder to your repository:
     - `index.html`
     - `notes.html`
     - `mcqs.html`
     - `css/styles.css`
     - `js/script.js`

4. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit: Add legal profession study portal"
   git push origin main
   ```

5. **Enable GitHub Pages**
   - Go to your repository settings
   - Scroll down to "GitHub Pages" section
   - Select `main` branch as the source
   - Click Save
   - Your site will be available at: `https://your-username.github.io/repository-name`
